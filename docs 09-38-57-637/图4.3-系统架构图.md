## 图 4.3 系统架构图

**对应小节**：4.3 系统架构设计  
**图注建议**：系统采用分层架构设计，由表示层、业务层、AI能力层和数据层构成；前后端支持本地图执行与代理后端双路径，并通过SSE实现AI任务实时反馈。

```mermaid
%%{init: {
  "theme": "base",
  "themeVariables": {
    "background": "#ffffff",
    "mainBkg": "#ffffff",
    "secondaryColor": "#ffffff",
    "tertiaryColor": "#ffffff",
    "primaryColor": "#ffffff",
    "primaryTextColor": "#000000",
    "primaryBorderColor": "#333333",
    "lineColor": "#333333",
    "clusterBkg": "#ffffff",
    "clusterBorder": "#cccccc",
    "titleColor": "#000000"
  },
  "flowchart": {
    "curve": "linear",
    "padding": 12,
    "nodeSpacing": 50,
    "rankSpacing": 50
  }
}}%%
flowchart TB
    U[求职用户]

    subgraph L1[表示层（UI）]
        direction LR
        A[Landing / Dashboard]
        B[简历编辑器]
        C[AI Lab界面]
        D[设置页]
    end

    subgraph L2[业务层（服务与状态）]
        direction LR
        E[前端业务逻辑\nHooks / Services]
        F[状态管理\nZustand Stores]
        G[路由与接口适配\n/api/node/* 与 /api/*]
    end

    subgraph L3[AI能力层（智能编排）]
        direction LR
        H[LangChain Chains]
        I[LangGraph Workflows\nResearch / Analyze / Rewrite]
        J[模型服务\nOpenAI / Anthropic / Google]
        K[检索增强\nTavily Search]
    end

    subgraph L4[数据层（持久化）]
        direction LR
        M[IndexedDB\nResume / Settings]
        N[运行时任务状态\n日志 / 阶段结果]
    end

    U --> A & B & C & D

    A & B & C & D --> E
    E --> F
    E --> G

    G --> H
    H --> I
    I --> J
    I --> K

    F --> M
    E --> N
    N -.SSE流式反馈.-> C

    classDef plain fill:#ffffff,stroke:#333333,color:#000000
    class U,A,B,C,D,E,F,G,H,I,J,K,M,N plain
    style L1 fill:#ffffff,stroke:#cccccc
    style L2 fill:#ffffff,stroke:#cccccc
    style L3 fill:#ffffff,stroke:#cccccc
    style L4 fill:#ffffff,stroke:#cccccc
```

## 使用说明

1. 打开 [Mermaid Live Editor](https://mermaid.live)。
2. 复制上方代码块（从 `%%{init` 到 `style L4` 行）。
3. 连线为折线/直线段（`curve: linear`），画布与子图为白底；虚线 `-.` 表示 SSE 流式反馈；导出 PNG 若背景非纯白，可用 SVG 后铺 `#ffffff`。
4. 若旧版 Mermaid 不支持 `U --> A & B & C & D` 或 `A & B & C & D --> E`，可拆成多行分别连接。
5. 点击 **Actions → PNG** 或 **SVG** 导出图片。
6. 插入论文并标注图号为「图 4.3 系统架构图」。
