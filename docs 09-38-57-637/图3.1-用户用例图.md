## 图 3.1 用户用例图

**对应小节**：3.2.1 用户用例分析  
**图注建议**：求职用户围绕简历管理、简历编辑、智能优化、职业规划、模拟面试和求职信生成完成全流程求职准备，系统形成一站式闭环服务。

```mermaid
%%{init: {
  "theme": "base",
  "themeVariables": {
    "background": "#ffffff",
    "mainBkg": "#ffffff",
    "secondaryColor": "#ffffff",
    "tertiaryColor": "#ffffff",
    "primaryColor": "#ffffff",
    "primaryTextColor": "#000000",
    "primaryBorderColor": "#333333",
    "lineColor": "#333333",
    "clusterBkg": "#ffffff",
    "clusterBorder": "#cccccc",
    "titleColor": "#000000"
  },
  "flowchart": {
    "curve": "linear",
    "padding": 12,
    "nodeSpacing": 50,
    "rankSpacing": 50
  }
}}%%
flowchart TB
    U[用户]
    subgraph ROW[" "]
        direction LR
        A[简历管理]
        B[个人中心/设置]
        C[查看岗位信息]
        D[简历编辑]
        E[模板切换与预览]
        F[简历智能优化]
        G[职业路径规划]
        J[模拟面试]
        K[求职信生成]
    end

    U --> A & B & C & D & E & F & G & J & K

    classDef whiteNode fill:#ffffff,stroke:#333333,color:#000000
    class U,A,B,C,D,E,F,G,J,K whiteNode
    style ROW fill:#ffffff,stroke:#cccccc
```

---

## 使用说明

1. 打开 [Mermaid Live Editor](https://mermaid.live)。
2. 复制上方代码块（从 `%%{init` 到最后一个 `style` 行）。
3. 粘贴后右侧生成图示；连线为 **折线/直线段**（`curve: linear`），画布与节点为 **白底**（`themeVariables.background` 等）。
4. 导出 PNG 时若背景仍非纯白，可在 Mermaid Live 的 **Actions** 中查看导出选项，或用 SVG 后在矢量软件中统一设背景为 `#ffffff`。
5. 若本地 Mermaid 版本较旧不支持 `U --> A & B & ...` 多线写法，可改为九行：`U --> A`、`U --> B` … `U --> K`。
6. 插入论文并标注图号为「图 3.1 用户用例图」。
