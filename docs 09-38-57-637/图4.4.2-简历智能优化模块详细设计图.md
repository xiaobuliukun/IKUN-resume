## 图 4.4.2 简历智能优化模块详细设计图

**对应小节**：4.4.2 简历智能优化模块详细设计  
**图注建议**：优化模块采用三阶段工作流（Research、Analyze、Rewrite），通过统一状态对象驱动节点流转，并以SSE向前端持续回传中间结果。

```mermaid
%%{init: {
  "theme": "base",
  "themeVariables": {
    "background": "#ffffff",
    "mainBkg": "#ffffff",
    "secondaryColor": "#ffffff",
    "tertiaryColor": "#ffffff",
    "primaryColor": "#ffffff",
    "primaryTextColor": "#000000",
    "primaryBorderColor": "#333333",
    "lineColor": "#333333",
    "clusterBkg": "#ffffff",
    "clusterBorder": "#cccccc",
    "titleColor": "#000000"
  },
  "flowchart": {
    "curve": "linear",
    "padding": 12,
    "nodeSpacing": 50,
    "rankSpacing": 50
  }
}}%%
flowchart TB
    U[用户输入JD并发起优化]
    RQ[前端请求层\nrunOptimization]
    ST[GraphState初始化\njd + resume]

    subgraph G1[Research阶段]
        R1[JD解析]
        R2[检索查询生成]
        R3[Web检索与反思循环]
        R4[研究结论输出]
        R1 --> R2 --> R3 --> R4
    end

    subgraph G2[Analyze阶段]
        A1[分析任务准备]
        A2[分类评估循环]
        A3[分析报告汇总]
        A1 --> A2 --> A3
    end

    subgraph G3[Rewrite阶段]
        W1[重写任务准备]
        W2[分区改写循环]
        W3[优化结果合并]
        W1 --> W2 --> W3
    end

    SSE[SSE流式回传\n节点状态/中间结果]
    OUT[优化简历输出并确认应用]

    U --> RQ --> ST --> G1 --> G2 --> G3 --> OUT
    G1 -.-> SSE
    G2 -.-> SSE
    G3 -.-> SSE
    SSE --> RQ

    classDef plain fill:#ffffff,stroke:#333333,color:#000000
    class U,RQ,ST,R1,R2,R3,R4,A1,A2,A3,W1,W2,W3,SSE,OUT plain
    style G1 fill:#ffffff,stroke:#cccccc
    style G2 fill:#ffffff,stroke:#cccccc
    style G3 fill:#ffffff,stroke:#cccccc
```

## 使用说明

1. 打开 [Mermaid Live Editor](https://mermaid.live)。
2. 复制上方代码块（从 `%%{init` 到 `style G3` 行）。
3. 连线为折线/直线段（`curve: linear`），画布与子图为白底；虚线 `-.` 表示各阶段向 SSE 推送；导出 PNG 若背景非纯白，可用 SVG 后铺 `#ffffff`。
4. 点击 **Actions → PNG** 或 **SVG** 导出图片。
5. 插入论文并标注图号为「图 4.4.2 简历智能优化模块详细设计图」。
