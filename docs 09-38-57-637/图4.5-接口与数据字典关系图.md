## 图 4.5 接口与数据字典关系图

**对应小节**：4.5 系统接口与数据字典概述  
**图注建议**：系统接口按 Research、Analyze、Rewrite 三阶段调用，围绕 Resume、Setting、GraphState 三类核心数据对象完成状态流转与结果输出。

```mermaid
%%{init: {
  "theme": "base",
  "themeVariables": {
    "background": "#ffffff",
    "mainBkg": "#ffffff",
    "secondaryColor": "#ffffff",
    "tertiaryColor": "#ffffff",
    "primaryColor": "#ffffff",
    "primaryTextColor": "#000000",
    "primaryBorderColor": "#333333",
    "lineColor": "#333333",
    "clusterBkg": "#ffffff",
    "clusterBorder": "#cccccc",
    "titleColor": "#000000"
  },
  "flowchart": {
    "curve": "linear",
    "padding": 12,
    "nodeSpacing": 50,
    "rankSpacing": 50
  }
}}%%
flowchart LR
    FE[前端调用层]

    subgraph API[接口层]
        direction TB
        R[/research/]
        A[/analyze/]
        W[/rewrite/]
    end

    subgraph DATA[数据对象层]
        direction TB
        D1[Resume]
        D2[Setting]
        D3[GraphState]
    end

    subgraph OUT[输出层]
        direction TB
        O1[SSE阶段状态]
        O2[analysisReport]
        O3[optimizedResume]
    end

    FE --> R & A & W

    R --> D1 & D2 & D3
    A --> D2 & D3
    W --> D2 & D3

    R --> O1
    A --> O1 & O2
    W --> O1 & O3

    classDef plain fill:#ffffff,stroke:#333333,color:#000000
    class FE,R,A,W,D1,D2,D3,O1,O2,O3 plain
    style API fill:#ffffff,stroke:#cccccc
    style DATA fill:#ffffff,stroke:#cccccc
    style OUT fill:#ffffff,stroke:#cccccc
```

---

## 使用说明

1. 打开 [Mermaid Live Editor](https://mermaid.live)。
2. 复制上方代码块（从 `%%{init` 到 `style OUT` 行）。
3. 连线为折线/直线段（`curve: linear`），画布与子图为白底；导出 PNG 若背景非纯白，可用 SVG 后铺 `#ffffff`。
4. 若旧版 Mermaid 不支持 `&` 多线写法，可拆成多行分别连接（与原文等价）。
5. 点击 **Actions → PNG** 或 **SVG** 导出图片。
6. 插入论文并标注图号为「图 4.5 接口与数据字典关系图」。
