## 图 4.1 系统功能模块图

**对应小节**：4.1 功能模块设计  
**图注建议**：系统采用“基础能力层 + 智能能力层 + 支撑能力层”三层模块结构，围绕简历编辑与岗位导向优化形成一站式求职辅助闭环。

```mermaid
%%{init: {
  "theme": "base",
  "themeVariables": {
    "background": "#ffffff",
    "mainBkg": "#ffffff",
    "secondaryColor": "#ffffff",
    "tertiaryColor": "#ffffff",
    "primaryColor": "#ffffff",
    "primaryTextColor": "#000000",
    "primaryBorderColor": "#333333",
    "lineColor": "#333333",
    "clusterBkg": "#ffffff",
    "clusterBorder": "#cccccc",
    "titleColor": "#000000"
  },
  "flowchart": {
    "curve": "linear",
    "padding": 12,
    "nodeSpacing": 50,
    "rankSpacing": 50
  }
}}%%
flowchart TB
    SYS[智能简历求职系统]

    subgraph L1[基础能力层]
        direction LR
        A[简历管理模块\n新建/复制/删除/保存]
        B[简历编辑模块\n信息编辑/模块排序/模板切换/预览]
    end

    subgraph L2[智能能力层]
        direction LR
        C[简历智能优化模块\n岗位解析/多维分析/分段重写]
        D[职业路径规划模块\n能力差距诊断/学习建议]
        E[模拟面试模块\n问题生成/回答反馈]
        F[求职信生成模块\n岗位定制文书生成]
    end

    subgraph L3[支撑能力层]
        direction LR
        G[设置与配置模块\n模型/参数/调用方式]
        H[状态与数据管理模块\nZustand + IndexedDB]
        I[流式通信模块\nSSE 实时反馈]
    end

    SYS --> L1 & L2 & L3

    A --> B
    B --> C
    C --> D
    C --> E
    C --> F

    G --> C
    H --> A
    H --> B
    H --> C
    I --> C
    I --> E
    I --> F

    classDef plain fill:#ffffff,stroke:#333333,color:#000000
    class SYS,A,B,C,D,E,F,G,H,I plain
    style L1 fill:#ffffff,stroke:#cccccc
    style L2 fill:#ffffff,stroke:#cccccc
    style L3 fill:#ffffff,stroke:#cccccc
```

## 使用说明

1. 打开 [Mermaid Live Editor](https://mermaid.live)。
2. 复制上方代码块（从 `%%{init` 到 `style L3` 行）。
3. 连线为折线/直线段（`curve: linear`），画布与子图为白底；导出 PNG 若背景非纯白，可用 SVG 后铺 `#ffffff`。
4. 若旧版 Mermaid 不支持 `SYS --> L1 & L2 & L3`，可拆成三行分别连接。
5. 点击 **Actions → PNG** 或 **SVG** 导出图片。
6. 插入论文并标注图号为「图 4.1 系统功能模块图」。
