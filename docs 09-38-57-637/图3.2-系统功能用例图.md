## 图 3.2 系统功能用例图

**对应小节**：3.2.2 系统功能用例图说明  
**图注建议**：系统围绕求职用户构建基础能力与智能能力两类用例，形成“简历管理-编辑-优化-求职准备”闭环流程。

```mermaid
%%{init: {
  "theme": "base",
  "themeVariables": {
    "background": "#ffffff",
    "mainBkg": "#ffffff",
    "secondaryColor": "#ffffff",
    "tertiaryColor": "#ffffff",
    "primaryColor": "#ffffff",
    "primaryTextColor": "#000000",
    "primaryBorderColor": "#333333",
    "lineColor": "#333333",
    "clusterBkg": "#ffffff",
    "clusterBorder": "#cccccc",
    "titleColor": "#000000"
  },
  "flowchart": {
    "curve": "linear",
    "padding": 12,
    "nodeSpacing": 50,
    "rankSpacing": 50
  }
}}%%
flowchart TB
    U[求职用户]

    subgraph BASE[基础功能用例]
        direction LR
        A[简历管理]
        B[个人中心与设置]
        C[查看岗位信息]
        D[简历编辑]
        E[模板切换与预览]
    end

    subgraph AI[智能功能用例]
        direction LR
        F[简历智能优化]
        G[职业路径规划]
        J[模拟面试]
        K[求职信生成]
    end

    U --> A & B & C & D & E & F & G & J & K

    A --> D
    B --> F
    C --> F
    D --> E
    D --> F
    F --> G
    F --> J
    F --> K

    classDef plain fill:#ffffff,stroke:#333333,color:#000000
    class A,B,C,D,E,F,G,J,K,U plain
    style BASE fill:#ffffff,stroke:#cccccc
    style AI fill:#ffffff,stroke:#cccccc
```

---

## 使用说明

1. 打开 [Mermaid Live Editor](https://mermaid.live)。
2. 复制上方代码块（从 `%%{init` 到 `style AI` 行）。
3. 连线为折线/直线段（`curve: linear`），画布与子图为白底；导出 PNG 若背景非纯白，可用 SVG 后在软件中铺 `#ffffff`。
4. 若旧版 Mermaid 不支持 `U --> A & B & ...`，可拆成多行 `U --> A`、`U --> B` …
5. 点击 **Actions → PNG** 或 **SVG** 导出图片。
6. 插入论文并标注图号为「图 3.2 系统功能用例图」。
