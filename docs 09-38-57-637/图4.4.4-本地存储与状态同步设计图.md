## 图 4.4.4 本地存储与状态同步设计图

**对应小节**：4.4.4 本地存储与状态同步设计  
**图注建议**：系统通过“运行时状态 + 本地持久化”双层机制实现数据同步、异常恢复与一致性保障。

```mermaid
%%{init: {
  "theme": "base",
  "themeVariables": {
    "background": "#ffffff",
    "mainBkg": "#ffffff",
    "secondaryColor": "#ffffff",
    "tertiaryColor": "#ffffff",
    "primaryColor": "#ffffff",
    "primaryTextColor": "#000000",
    "primaryBorderColor": "#333333",
    "lineColor": "#333333",
    "clusterBkg": "#ffffff",
    "clusterBorder": "#cccccc",
    "titleColor": "#000000"
  },
  "flowchart": {
    "curve": "linear",
    "padding": 12,
    "nodeSpacing": 50,
    "rankSpacing": 50
  }
}}%%
flowchart TB
    U[用户编辑操作]
    FE[前端编辑器页面]
    ST[运行时状态层\nZustand Stores]
    CK{数据是否变更}
    DB[本地持久化层\nIndexedDB]
    OK[保存成功反馈]
    ER[异常提示与重试]
    LD[页面重进或刷新]
    RS[数据恢复与状态重建]
    UI[界面回显与继续编辑]

    U --> FE --> ST --> CK
    CK -- 是 --> DB --> OK --> UI
    CK -- 否 --> UI
    DB -- 写入失败 --> ER --> ST
    LD --> DB --> RS --> ST --> UI

    classDef plain fill:#ffffff,stroke:#333333,color:#000000
    class U,FE,ST,CK,DB,OK,ER,LD,RS,UI plain
```

---

## 使用说明

1. 打开 [Mermaid Live Editor](https://mermaid.live)。
2. 复制上方代码块（从 `%%{init` 到 `class ... plain` 行）。
3. 连线为折线/直线段（`curve: linear`），画布为白底；导出 PNG 若背景非纯白，可用 SVG 后铺 `#ffffff`。
4. 点击 **Actions → PNG** 或 **SVG** 导出图片。
5. 插入论文并标注图号为「图 4.4.4 本地存储与状态同步设计图」。
